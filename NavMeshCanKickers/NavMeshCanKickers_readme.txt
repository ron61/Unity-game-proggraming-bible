===========================================
「Unity ゲーム プログラミング・バイブル」サンプルプログラム

No.08 NavMesh Can Kickers
===========================================

【フォルダ/ファイルの説明】
・NavMeshCanKickers
ナビゲーションシステムを使用したアクションゲーム Can Kickers の Unity プロジェクトフォルダです。

・NavMeshCanKickers/

・AssetBundleCanKickers.exe
アプリのWindows 10用実行ファイルです。ダブルクリックでプレイできます。

・NavMeshCanKickers_readme.txt
このファイルです。

【サンプルアプリの使い方】
タイトル画面から始まります。画面をタッチ/クリックするとゲーム開始です。

- フィールドに次々と現れる缶を蹴るゲームです。
- 缶には得点があります。一定時間内に一番得点すれば勝利です。お茶缶は高得点です。
- フィールドにはプレーヤー操作キャラの他に、二人の敵プレーヤーキャラがいて、自動でプレイします。
- プレーヤー以外のキャラクタは、自動で缶を目指して移動しながら缶を蹴ります。
- フィールド上の木箱は、何度か蹴ると壊れてアイテムが出ます。スピードアップや金縛り等の効果があります。

操作方法は次のようになっています。

- プレーヤーをマウスでフィールドをポイントし、クリックすることでその場所へキャラクタを移動させられます。
- ボタンホールドしながらマウスを動かすと(ドラッグ)、ポイント位置を追いかける方に移動します。
- 右クリックかZキーでポイントしている方向に向かって蹴りアクションをします。

フィールド中の平らな「床」に乗っていればキャラクタは走って移動可能です。
床がつながっていなくて走って移動ができない場所もありますが、
キャラクタはそのような場所を自動で「ジャンプ」して移動できるようになっています。
双方向にジャンプできるところもあれば、高低差のためにて高架ジャンプしかできない場所もあります。

【利用上の注意点】
- プロジェクトには No.12 ノベルゲームで使用している TextMesh Proが NavMeshCanKickers/Assets/TextMesh Pro として含まれています。
- Unity エディタ上でプレイする際は、Title シーンや Game シーンを開いて再生してください。
